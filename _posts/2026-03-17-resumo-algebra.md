## Sobre o projeto
Este projeto implementa uma **rede neural artificial do tipo Perceptron** em Python. Ele serve como estudo de conceitos de aprendizado supervisionado e funções de ativação simples.

### Funcionalidades
- Treinamento de um Perceptron para classificação binária;
- Função de ativação passo (step function);
- Testes com diferentes conjuntos de dados.

### Como usar
Clone o repositório:

```
git clone https://github.com/natamleao/Perceptron-Neural-Network.git
````

Entre na pasta e execute o script principal em Python:

```
python perceptron.py
```
